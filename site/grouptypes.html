Group types are not part of the VOOT specification.

Still this page contains an work in progress example of how to define group types.

Group types

eduOrg

{% highlight json %} { "id": "eduperson:org", "displayName": { "en": "Organization", "nb": "Organisasjon" } } {% endhighlight %}

The following properties are optional.
